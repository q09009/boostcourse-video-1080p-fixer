# 🚀 Boostcourse Video 1080P Fixer

> 부스트코스(Boostcourse) 강의 영상을 시청할 때마다 번거롭게 화질을 변경하시나요?  
> 이 익스텐션은 영상이 재생되는 순간 **자동으로 최대 화질(1080p)로 고정**해줍니다.

---

## ✨ 주요 기능
- **자동 화질 고정**: 영상 재생 버튼을 누르는 순간 자동으로 설정 메뉴를 열어 1080p 화질을 선택합니다.
- **스마트 감지**: `video` 태그의 재생 상태를 감시하여, 일시정지 후 재시행 시에는 불필요하게 작동하지 않고 **페이지 로드/강의 전환 시에만** 정확히 작동합니다.
- **사용자 설정**: 팝업창의 슬라이더를 통해 화질 설정이 시작될 **대기 시간(Delay)**을 밀리초(ms) 단위로 조절할 수 있습니다.
- **설정 기억**: 한 번 설정한 대기 시간은 브라우저에 저장되어 다음 접속 시에도 유지됩니다.

## 🛠️ 설치 방법 (개발자 모드)
현재 웹스토어 심사 전이므로 아래 방법으로 수동 설치가 가능합니다.

1. 이 저장소를 `Zip` 파일로 다운로드하거나 `git clone` 합니다.
2. 크롬 브라우저에서 `chrome://extensions/` 주소로 이동합니다.
3. 우측 상단의 **'개발자 모드'**를 활성화합니다.
4. 왼쪽 상단의 **'압축해제된 확장 프로그램을 로드합니다'** 버튼을 클릭합니다.
5. 다운로드한 프로젝트 폴더를 선택합니다.

## ⚙️ 사용법
1. 부스트코스 강의 페이지에 접속합니다.
2. 익스텐션 아이콘을 클릭하여 원하는 **설정 대기 시간**을 조절합니다. (기본값: 1000ms)
3. 영상을 재생하면 자동으로 화질이 변경되는 것을 확인하세요!
